What is an algebraic expression?
Algebraic expression = a combination of numbers, variables (letters like x, y, a), and operations (+, −, ×, ÷).
No equal sign! (that's an equation).
Variable: a letter representing an unknown value Coefficient: number multiplied by a variable Constant: fixed number (e.g. 5, -2) Terms: parts separated by + or -

Example 1: Real life (Shule yetu)

Neema buys 3 notebooks and 2 pens. Each notebook costs sh 1,500 and each pen costs sh 500. But what if we don't know the price? Let variable n = price per notebook, p = price per pen.
3n + 2p is an algebraic expression!

Example 2: Terms & Coefficients

Expression: 5x² - 3xy + 7
➜ Terms: 5x² , -3xy , +7
➜ Coefficient of x² is 5, coefficient of xy is -3, constant = 7.
✨ Variables can be more than one letter!

From me (Neema): “In Tanzania, when we farm maize or sell groceries, algebraic expressions help us calculate profits without fixed numbers. Imagine: '2 sacks of maize + 3 buckets of beans' = 2m + 3b. Simple, right?”
